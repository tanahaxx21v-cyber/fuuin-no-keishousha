## ゲーム概要

| 項目 | 内容 |
|---|---|
| ジャンル | ブラウザRPG（パワポケ4 裏サクセス準拠） |
| プレイ時間 | 約60〜90分（Normal） |
| 舞台 | ルミナ大陸（21ロケーション） |
| 目的 | 100日以内に3つの封印石を集め、砂漠遺跡の終末記録体アーカイブを倒す |
| クリア目標レベル | **Lv20前後** |

## スタート〜クリアまでの基本ルート

```
アルセリア王都（スタート）
 ↓ 旅人の宿 or 東関所
ガルド皆都市 → 廃鉱山（炎の封印石）
 ↓
竜の峠（嵐の封印石）
 ↓
ベルン → 盗賊アジト
 ↓
エルナの里 → 精霊の泉 → 古代神殿（闇の封印石）
 ↓
サハル → 砂漠遺跡（最終決戦）
```

### EXP計算式
- 次レベルまでのEXP = `現在レベル × 15`
- 例: Lv5→Lv6 = 75EXP, Lv19→Lv20 = 285EXP
- **Lv20到達に必要な総EXP: 2,850**

## マップ & 経路

### 21ロケーション構成

**6つの町（宿屋・ショップあり）**
- アルセリア王都 / ベルン商業都市 / サハル砂漠都市
- ミレア港町 / エルナの里 / ガルド皆都市

**10の中間地点（移動エンカウントあり）**
- 旅人の宿 / 東関所 / 大橋 / 川辺の村 / 見張り塔
- 灯台岬 / 精霊の泉 / 交易所 / 海岸街道 / 森の入口

**5つのダンジョン・特殊場所**
- 廃鉱山（炎の封印石） / 竜の峠（嵐の封印石） / 盗賊アジト
- 古代神殿（闇の封印石） / **砂漠遺跡（最終決戦・封印石3つ必要）**

### 移動エンカウントシステム
- 中間地点（relay）を通過するとき：
  - **30%**: 敵エンカウント
  - **15%**: ランダムイベント（ゴールド発見・ポーション入手・HP回復）
  - **55%**: 何も起きない

## バトルシステム

### 基本
- ターン制コマンドバトル（速度順）
- 行動：通常攻撃 / スキル / アイテム / 逃げる（ボス戦不可）
- ボスからは逃げられない

### MP切れ時
- MPが不足した状態でスキルを使おうとすると**1ターン休憩**
- 休憩中はMP最大値の8%（最低5）を回復

### 状態異常
| 名前 | 効果 | 持続 |
|---|---|---|
| 毒 | 毎ターン8ダメージ | 4ターン |
| スタン | 行動不能 | 2ターン |
| 攻撃UP | ダメージ1.5倍 | 3ターン |
| 防御UP | 被ダメージ0.6倍 | 3ターン |

## おすすめパーティ編成

### バランス型（初心者向け）
- **ガレス** + **リズ** + **ノア**
- 回復・防御・アタッカーの三角形。安定して進める

### 速攻型（上級者向け）
- **ヴァイス** + **ノア** + **エルク**
- 速度が高い3人で先手を取り続ける。回復手段がないため要腕前

### 魔法パーティ
- **セシル** + **イリス** + **リズ**
- 全体魔法連発で雑魚をまとめて倒す。ボス戦はMP管理が重要

### 最強パーティ（ゼノ加入後）
- **ゼノ** + **リズ** + **ローガン** or **エルク**
- ゼノの圧倒的な攻撃力でゴリ押し可能

## セーブ機能
- ゲームは**自動セーブ**（戦闘終了・移動などのタイミングで自動保存）
- ステータスバーの💾ボタンで手動セーブも可能
- ブラウザのlocalStorageに保存（ブラウザをクリアすると消える）
